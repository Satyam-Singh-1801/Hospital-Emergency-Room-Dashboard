# Hospital-Emergency-Room-Dashboard
## Recommended Structure and Order
###🚀 Purpose of the Project

The primary goal of this project is to:

Monitor emergency room performance
Analyze patient management and operational efficiency
Track important healthcare KPIs
Identify peak patient traffic hours
Improve decision-making through data visualization
###🛠️ Tech Stack
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization
Microsoft Excel / CSV Dataset
📂 Data Source

The dataset used in this project contains hospital emergency room records including:

Patient Details
Admission Status
Wait Time
Department Referrals
Patient Demographics
Satisfaction Scores
Visit Date & Time

The data was cleaned, transformed, and modeled inside Power BI using Power Query and DAX.
